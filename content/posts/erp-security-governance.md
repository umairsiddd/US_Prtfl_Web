## Project Overview

Designed and implemented a comprehensive security governance dashboard for an international enterprise client, providing real-time visibility into user roles, duties, and licensing utilization across Microsoft Dynamics 365 Finance & Operations.

## Challenge

The client had no centralized mechanism to monitor ERP user access, licensing consumption, or segregation of duties compliance. This created significant audit risk and unnecessary licensing overhead.

## Solution

- Architected a custom **Security Roles & License Management Dashboard** within D365 F&O
- Configured granular security roles, duties, privileges, and access controls
- Built SSRS reports providing real-time licensing utilization analytics
- Documented the security framework in a comprehensive FDD/TDD

## Technologies Used

`Microsoft Dynamics 365 F&O` `X++` `SSRS` `Azure DevOps` `SQL`

## Business Impact

- Enabled executive leadership to **optimize licensing costs** across the organization
- Ensured **100% compliance** with internal audit standards and data security protocols
- Established a scalable governance framework adaptable to future organizational growth
